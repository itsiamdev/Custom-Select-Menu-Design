# Custom Select Menu Design

A custom-designed select menu for choosing social media platforms. This project features a dropdown interface with icons for popular social media sites including Facebook, Twitter, Instagram, YouTube, Pinterest, and WhatsApp.

Built with HTML, CSS, and JavaScript for a smooth, interactive user experience.

## Live Demo

Check out the live demo: [https://iam269.github.io/Custom-Select-Menu-Design/](https://iam269.github.io/Custom-Select-Menu-Design/)

## Features

- Custom dropdown select menu
- Social media platform icons
- Responsive design
- Smooth animations

## Usage

Simply open `index.html` in a web browser to view and interact with the custom select menu.
